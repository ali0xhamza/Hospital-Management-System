# Hospital Management System (DBMS Project)
# Project Overview

This project is a Hospital Management System built using PHP and SQL. It allows the management of patients, doctors, appointments, and billing. The system provides an easy-to-use interface for hospital administration tasks.

# Folder Structure
appointment_management/      # Handles appointments (add & view)
assets/                      # Contains CSS and images
billing/                     # Handles billing (bill files)
doctor_management/           # CRUD operations for doctors
patient_management/          # CRUD operations for patients
dashboard.php                # Main dashboard of the system
db.php                       # Database connection file
header.php                   # Header template
hospital_db.sql              # Database SQL file
hospital1.jpg                # Sample hospital image
image.jpg                    # Additional image
index.php                    # Homepage
login.php                    # Login page
logout.php                   # Logout page
# Features
# Appointment Management

* Add new appointments

* View existing appointments

# Doctor Management

* Add, edit, delete, and view doctor records

# Patient Management

* Add, edit, delete, and view patient records

# Billing

* Generate and view patient bills

# Assets

* CSS for styling

* Images used in the system

# Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ali0xhamza/Hospital-Management-System.git

2. Import the hospital_db.sql into your MySQL database.

3. Update db.php with your database credentials.

4. Run the project on a local server (XAMPP/WAMP/LAMP).

5. Open index.php in your browser.

# Usage

1. Login with your credentials.

2. Manage doctors, patients, appointments, and billing through the dashboard.

3. Logout after completing your session.
